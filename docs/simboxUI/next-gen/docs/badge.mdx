# Badge

The Badge component is used to display small status or notification indicators.

## Usage

```tsx
import Badge from './Badge';

<Badge variant="success">Success</Badge>
<Badge variant="danger" size="lg" pill>Error</Badge>
<Badge color="indigo" icon={<CheckIcon />}>Approved</Badge>
```

## Props

| Prop      | Type                                                                 | Default     | Description                                                          |
| --------- | -------------------------------------------------------------------- | ----------- | -------------------------------------------------------------------- |
| children  | React.ReactNode                                                      | -           | The content to display inside the badge.                             |
| variant   | 'primary' \| 'secondary' \| 'success' \| 'danger' \| 'warning' \| 'info' | 'primary'   | The variant of the badge.                                            |
| size      | 'sm' \| 'md' \| 'lg'                                                   | 'md'        | The size of the badge.                                               |
| pill      | boolean                                                              | false       | Whether the badge should have rounded corners (pill shape).          |
| icon      | React.ReactNode                                                      | -           | An optional icon to display before the badge content.                |
| color     | string                                                               | -           | A custom background color for the badge (e.g., 'indigo', 'purple'). |
| className | string                                                               | ''          | Additional CSS classes to apply to the badge.                        |

## Variants

The `variant` prop allows you to choose from predefined color variants:

- `primary` (default)
- `secondary`
- `success`
- `danger`
- `warning`
- `info`

## Sizes

The `size` prop allows you to control the size of the badge:

- `sm` - Small
- `md` (default) - Medium
- `lg` - Large

## Pill Shape

Setting the `pill` prop to `true` will give the badge rounded corners, creating a pill-like shape.

## Custom Colors

You can specify a custom background color for the badge using the `color` prop. Pass the color name as a string (e.g., 'indigo', 'purple'), and the component will apply the corresponding Tailwind color classes.

## Icon Support

The `icon` prop allows you to pass an icon component to be displayed before the badge content. The icon will be rendered with a right margin.

## Dark Mode

The badge component automatically adapts to dark mode using Tailwind's dark mode classes. The background colors will be adjusted to provide better contrast in dark mode.
