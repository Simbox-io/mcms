# Radio

The Radio component is a customizable radio button input with label and optional description.

## Usage

```tsx
import Radio from './Radio';

<Radio
  label="Option 1"
  value="option1"
  checked={selectedOption === 'option1'}
  onChange={handleOptionChange}
  description="Additional information about Option 1"
/>
```

## Props

| Prop        | Type                   | Default | Description                                                 |
| ----------- | ---------------------- | ------- | ----------------------------------------------------------- |
| label       | string                 | -       | The label text for the radio button.                        |
| value       | string                 | -       | The value associated with the radio button.                 |
| checked     | boolean                | -       | Indicates whether the radio button is checked.              |
| onChange    | (value: string) => void | -       | Callback function to handle the change event.               |
| disabled    | boolean                | false   | Determines if the radio button is disabled.                 |
| className   | string                 | ''      | Additional CSS classes to apply to the radio button.        |
| description | string                 | ''      | Optional description text to provide additional information. |

## Additional Features

- Supports dark mode using Tailwind's `dark` variant.
- Includes subtle animations using Framer Motion for improved user experience.
- Allows providing an optional description for each radio option.
