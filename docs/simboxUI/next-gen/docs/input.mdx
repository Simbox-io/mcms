# Input

The Input component is a customizable and enhanced input field with support for various input types, labels, error handling, and password visibility toggle.

## Usage

```tsx
import Input from './Input';

<Input
  type="text"
  value={inputValue}
  onChange={handleInputChange}
  placeholder="Enter your name"
  label="Name"
  error={inputError}
/>
```

## Props

| Prop        | Type                                      | Default | Description                                                                                  |
| ----------- | ----------------------------------------- | ------- | -------------------------------------------------------------------------------------------- |
| type        | 'text' \| 'number' \| 'email' \| 'password' | 'text'  | The type of the input field.                                                                 |
| value       | string                                    | -       | The current value of the input field.                                                        |
| onChange    | (value: string) => void                   | -       | The callback function to handle input value changes.                                         |
| placeholder | string                                    | ''      | The placeholder text for the input field.                                                    |
| disabled    | boolean                                   | false   | Determines if the input field is disabled.                                                   |
| className   | string                                    | ''      | Additional CSS classes to apply to the input field.                                          |
| label       | string                                    | ''      | The label text to display above the input field.                                             |
| error       | string                                    | ''      | The error message to display below the input field.                                          |

## Additional Features

- **Password Visibility Toggle**: When the input type is set to 'password', a button with an eye icon is displayed to toggle the visibility of the password. Clicking the button switches between masked and unmasked password views.

- **Error Handling**: If an error message is provided via the `error` prop, it will be displayed below the input field in red color to indicate an error state.

- **Dark Mode Support**: The component is styled using Tailwind CSS classes and supports dark mode out of the box. It automatically adapts to the dark mode theme when enabled.

- **Accessibility**: The component includes focus styles to provide visual feedback when the input field is focused, improving accessibility for keyboard users.

## Dependencies

The component utilizes the following dependencies:

- `@heroicons/react/solid`: Provides the eye icon for the password visibility toggle button.

Make sure to install the required dependencies before using the Input component.
