# ProgressBar

The ProgressBar component displays a progress bar with customizable appearance and optional label.

## Usage

```tsx
import ProgressBar from './ProgressBar';

<ProgressBar value={75} max={100} label="Progress" />
```

## Props

| Prop               | Type      | Default         | Description                                                           |
| ------------------ | --------- | --------------- | --------------------------------------------------------------------- |
| value              | number    | -               | The current value of the progress bar.                                |
| max                | number    | 100             | The maximum value of the progress bar.                                |
| label              | string    | ''              | The label text to display alongside the progress bar.                 |
| showPercentage     | boolean   | true            | Whether to display the percentage value alongside the label.          |
| showValue          | boolean   | false           | Whether to display the current value and max value alongside the label. |
| className          | string    | ''              | Additional CSS classes to apply to the progress bar container.        |
| barColor           | string    | 'bg-blue-500'   | The color of the progress bar.                                        |
| barBackgroundColor | string    | 'bg-gray-200'   | The background color of the progress bar.                             |
| labelColor         | string    | 'text-gray-600' | The color of the label text.                                          |
| height             | string    | 'h-2.5'         | The height of the progress bar.                                       |
| animate            | boolean   | true            | Whether to animate the progress bar when the value changes.           |

## Additional Customization

- Use the `className` prop to add custom CSS classes to the progress bar container for further styling.
- Customize the colors of the progress bar, background, and label text using the `barColor`, `barBackgroundColor`, and `labelColor` props.
- Adjust the height of the progress bar using the `height` prop with a valid Tailwind height class.
- Enable or disable the animation of the progress bar using the `animate` prop.

## Dark Mode Support

The ProgressBar component supports dark mode out of the box. It uses Tailwind's dark mode classes to automatically adjust the colors in dark mode. Ensure that you have set up dark mode in your Tailwind configuration for this feature to work.
