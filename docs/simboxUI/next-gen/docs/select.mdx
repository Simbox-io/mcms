# Select

The Select component is an enhanced dropdown component that allows users to select an option from a list of options.

## Usage

```tsx
import Select from './Select';

const options = [
  { value: 'option1', label: 'Option 1' },
  { value: 'option2', label: 'Option 2' },
  { value: 'option3', label: 'Option 3' },
];

const [selectedValue, setSelectedValue] = useState('');

<Select
  options={options}
  value={selectedValue}
  onChange={(value) => setSelectedValue(value)}
  placeholder="Select an option"
  disabled={false}
  className="w-64"
/>
```

## Props

| Prop        | Type                  | Default           | Description                                                   |
| ----------- | --------------------- | ----------------- | ------------------------------------------------------------- |
| options     | Option[]              | -                 | An array of options to display in the select dropdown.        |
| value       | string                | -                 | The currently selected value.                                 |
| onChange    | (value: string) => void | -                 | Callback function to handle the change event when an option is selected. |
| placeholder | string                | 'Select an option' | Placeholder text to display when no option is selected.       |
| disabled    | boolean               | false             | Determines whether the select is disabled.                    |
| className   | string                | ''                | Additional CSS classes to apply to the select component.      |

## Option Interface

The `Option` interface defines the shape of each option object in the `options` array:

```tsx
interface Option {
  value: string;
  label: string;
}
```

- `value`: The unique value associated with the option.
- `label`: The display text for the option.

## Features

- Improved accessibility with keyboard navigation support.
- Enhanced styling using Tailwind CSS classes.
- Dropdown menu closes when clicking outside the component.
- Customizable placeholder text.
- Disabled state support.
- Dark mode compatible.

## Dependencies

The enhanced Select component relies on the following dependencies:

- `@headlessui/react`: Provides accessible UI components, such as the `Listbox` used in this component.
- `@heroicons/react`: Provides icons, such as the `CheckIcon` and `ChevronDownIcon` used in this component.

Make sure to install these dependencies before using the Select component.
