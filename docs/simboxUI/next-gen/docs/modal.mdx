# Modal

The Modal component is a reusable and customizable modal dialog for displaying content on top of the current page.

## Usage

```tsx
import Modal from './Modal';

const [isOpen, setIsOpen] = useState(false);

const handleOpen = () => setIsOpen(true);
const handleClose = () => setIsOpen(false);

return (
  <>
    <button onClick={handleOpen}>Open Modal</button>
    <Modal
      isOpen={isOpen}
      onClose={handleClose}
      title="Modal Title"
    >
      <p>Modal content goes here...</p>
    </Modal>
  </>
);
```

## Props

| Prop                 | Type                 | Default     | Description                                                 |
| -------------------- | -------------------- | ----------- | ----------------------------------------------------------- |
| `isOpen`             | `boolean`            | -           | Determines whether the modal is open or closed.           |
| `onClose`            | `() => void`         | -           | The callback function to invoke when the modal is closed. |
| `title`              | `string`             | -           | The title to display in the modal header.                |
| `children`           | `React.ReactNode`    | -           | The content to display inside the modal.                 |
| `className`          | `string`             | `''`        | Additional CSS classes to apply to the modal.            |
| `overlayClassName`   | `string`             | `''`        | Additional CSS classes to apply to the modal overlay.    |
| `closeOnOverlayClick`| `boolean`            | `true`      | Whether to close the modal when the overlay is clicked.  |
| `closeOnEscape`      | `boolean`            | `true`      | Whether to close the modal when the Escape key is pressed. |

## Functionality

- The Modal component is rendered as a portal, meaning it is rendered outside of the normal DOM hierarchy and displayed on top of the current page.
- The modal can be opened and closed using the `isOpen` prop and the `onClose` callback function.
- The modal can be closed by clicking on the overlay or by pressing the Escape key (controlled by the `closeOnOverlayClick` and `closeOnEscape` props).
- The modal content is wrapped in a `motion.div` component from the `framer-motion` library, which provides smooth animations for the modal's appearance and disappearance.

## Styling

- The modal and overlay have default styles applied using Tailwind CSS classes.
- The `className` and `overlayClassName` props can be used to apply additional CSS classes for custom styling.
- The modal content is styled with a white background, rounded corners, and a drop shadow.
- The modal header includes a title and a close button.
- The modal content area has padding applied.
