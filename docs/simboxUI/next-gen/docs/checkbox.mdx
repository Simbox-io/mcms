# Checkbox

The Checkbox component is a customizable and accessible checkbox input with label.

## Usage

```tsx
import Checkbox from './Checkbox';

<Checkbox
  label="Remember me"
  checked={isChecked}
  onChange={handleCheckboxChange}
  disabled={isDisabled}
  indeterminate={isIndeterminate}
/>
```

## Props

| Prop          | Type                        | Default | Description                                                                         |
| ------------- | --------------------------- | ------- | ----------------------------------------------------------------------------------- |
| label         | string                      | -       | The label text for the checkbox.                                                    |
| checked       | boolean                     | -       | The checked state of the checkbox.                                                  |
| onChange      | (checked: boolean) => void  | -       | The callback function to handle checkbox state changes.                             |
| disabled      | boolean                     | false   | Whether the checkbox is disabled.                                                   |
| className     | string                      | ''      | Additional CSS classes to apply to the checkbox container.                          |
| indeterminate | boolean                     | false   | Whether the checkbox is in an indeterminate state.                                  |

## Customization

The Checkbox component supports customization through the `className` prop. You can pass additional CSS classes to style the checkbox container as needed.

The component also supports dark mode out of the box. The label text color automatically adjusts based on the dark mode using Tailwind CSS classes.

## Accessibility

The Checkbox component is built with accessibility in mind. It uses the appropriate HTML input type and associates the label with the checkbox input. The checkbox is focusable and provides visual feedback on focus using a prominent focus ring style.

## Animation

The Checkbox component incorporates subtle animations using the `framer-motion` library. The checkbox scales slightly on hover and tap, providing visual feedback to the user. The label fades in and slides from the left when the component mounts, adding a smooth transition effect.
