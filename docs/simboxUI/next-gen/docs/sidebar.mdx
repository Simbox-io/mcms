# Sidebar

The Sidebar component is a navigation menu that can be toggled open and closed. It displays a list of navigation items with icons and labels.

## Usage

```tsx
import Sidebar from './Sidebar';

const sidebarItems = [
  {
    label: 'Home',
    href: '/',
    icon: <HomeIcon />,
  },
  {
    label: 'About',
    href: '/about',
    icon: <InfoIcon />,
  },
  {
    label: 'Contact',
    href: '/contact',
    icon: <MailIcon />,
  },
];

<Sidebar items={sidebarItems} />
```

## Props

| Prop      | Type         | Default | Description                                                 |
| --------- | ------------ | ------- | ----------------------------------------------------------- |
| items     | SidebarItem[] | -       | An array of sidebar items containing label, href, and icon. |
| className | string       | ''      | Additional CSS classes to apply to the sidebar container.   |

## SidebarItem Interface

The `SidebarItem` interface defines the shape of each sidebar item object in the `items` array:

```tsx
interface SidebarItem {
  label: string;
  href: string;
  icon: React.ReactNode;
}
```

- `label`: The text label for the sidebar item.
- `href`: The URL or path to navigate to when the item is clicked.
- `icon`: The icon component to display alongside the label.

## Features

- Smooth open and close animations using `framer-motion`.
- Automatically closes the sidebar when an item is clicked.
- Supports dark mode styling using Tailwind CSS.
- Responsive design that adapts to different screen sizes.

## Customization

- Customize the sidebar width by modifying the `w-64` class in the sidebar container.
- Adjust the colors and styles using Tailwind CSS classes or by overriding the default styles.
- Add additional content or sections to the sidebar as needed.

## Dependencies

- `react`
- `react-router-dom`
- `framer-motion`
- `tailwindcss`

Make sure to have these dependencies installed in your project before using the Sidebar component.
