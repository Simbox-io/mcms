# Pagination

The Pagination component provides a customizable and interactive pagination navigation.

## Usage

```tsx
import Pagination from './Pagination';

<Pagination
  currentPage={3}
  totalPages={10}
  onPageChange={(page) => console.log(`Navigating to page ${page}`)}
  className="mt-4"
  siblingCount={1}
  boundaryCount={2}
/>
```

## Props

| Prop          | Type                      | Default | Description                                                           |
| ------------- | ------------------------- | ------- | --------------------------------------------------------------------- |
| currentPage   | number                    | -       | The current active page number.                                       |
| totalPages    | number                    | -       | The total number of pages.                                            |
| onPageChange  | (page: number) => void    | -       | Callback function invoked when a page number is clicked.              |
| className     | string                    | ''      | Additional CSS classes to apply to the pagination container.          |
| siblingCount  | number                    | 1       | The number of page numbers to show on each side of the current page.  |
| boundaryCount | number                    | 1       | The number of page numbers to show at the beginning and end.          |

## Customization

The Pagination component can be customized using the following props:

- `className`: Allows passing additional CSS classes to the pagination container for custom styling.
- `siblingCount`: Determines the number of page numbers to show on each side of the current page. Default is 1.
- `boundaryCount`: Determines the number of page numbers to show at the beginning and end of the pagination. Default is 1.

## Accessibility

The Pagination component follows accessibility best practices:

- Uses semantic HTML elements like `<nav>` and `<button>` for better screen reader support.
- Provides `aria-label` attribute to the `<nav>` element to describe the pagination navigation.
- Disables the previous and next buttons when on the first and last pages, respectively.

## Styling

The Pagination component utilizes Tailwind CSS classes for consistent and responsive styling. It supports dark mode by applying dark mode-specific classes.

The component includes hover and focus styles to enhance interactivity and visual feedback.

## Icons

The Pagination component uses chevron icons from the Heroicons library for the previous and next page navigation buttons.
