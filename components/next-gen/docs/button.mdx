# Button

The Button component is a versatile and customizable button with various options for styling, behavior, and functionality.

## Usage

```tsx
import Button from './Button';

<Button variant="primary" size="medium" onClick={() => console.log('Button clicked')}>
  Click me
</Button>
```

## Props

| Prop             | Type                                           | Default   | Description                                                                         |
| ---------------- | ---------------------------------------------- | --------- | ----------------------------------------------------------------------------------- |
| children         | React.ReactNode                                | -         | The content to be rendered inside the button.                                       |
| onClick          | () => void                                     | -         | The callback function to be invoked when the button is clicked.                     |
| variant          | 'primary' \| 'secondary' \| 'outline' \| 'text' \| 'link' | 'primary' \| 'danger' | The variant of the button that determines its appearance.                           |
| size             | 'small' \| 'medium' \| 'large'                   | 'medium'  | The size of the button.                                                            |
| disabled         | boolean                                        | false     | Whether the button is disabled.                                                     |
| loading          | boolean                                        | false     | Whether the button is in a loading state.                                           |
| fullWidth        | boolean                                        | false     | Whether the button should take up the full width of its container.                  |
| className        | string                                         | ''        | Additional CSS classes to be applied to the button.                                 |
| icon             | React.ReactNode                                | -         | An icon to be rendered before the button text.                                      |
| tooltipContent   | string                                         | -         | The content to be displayed in the tooltip when hovering over the button.           |
| tooltipPlacement | 'top' \| 'right' \| 'bottom' \| 'left'            | 'top'     | The placement of the tooltip relative to the button.                                |
| href             | string                                         | -         | The URL to navigate to when the button is clicked. If provided, renders a link.     |
| target           | '_blank' \| '_self' \| '_parent' \| '_top'        | '_self'   | The target attribute specifying where to open the linked URL.                       |

## Variants

The Button component supports the following variants:

- `primary`: A primary button with a blue background color.
- `secondary`: A secondary button with a gray background color.
- `outline`: An outline button with a transparent background and a border.
- `text`: A text button with no background color and blue text.
- `link`: A link button with blue underlined text.
- 'danger': A button with a red background color.

## Sizes

The Button component supports the following sizes:

- `small`: A small-sized button.
- `medium`: A medium-sized button (default).
- `large`: A large-sized button.

## Additional Props

- `disabled`: Disables the button and applies a visual disabled style.
- `loading`: Indicates a loading state and displays a spinning loading indicator.
- `fullWidth`: Makes the button take up the full width of its container.
- `className`: Allows passing additional CSS classes to the button for custom styling.
- `icon`: Renders an icon before the button text.
- `tooltipContent`: Displays a tooltip with the specified content when hovering over the button.
- `tooltipPlacement`: Specifies the placement of the tooltip relative to the button.
- `href`: Renders the button as a link with the specified URL.
- `target`: Specifies where to open the linked URL.
