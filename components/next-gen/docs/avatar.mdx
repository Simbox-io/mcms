# Avatar

The Avatar component displays a user's profile picture or a placeholder image.

## Usage

```tsx
import Avatar from './Avatar';

<Avatar src="/path/to/image.jpg" alt="User Avatar" size="medium" />
```

## Props

| Prop      | Type                             | Default   | Description                                         |
| --------- | -------------------------------- | --------- | --------------------------------------------------- |
| src       | string                           | -         | The URL of the avatar image.                        |
| alt       | string                           | -         | The alternative text for the avatar image.          |
| size      | 'small' \| 'medium' \| 'large'   | 'medium'  | The size of the avatar.                             |
| className | string                           | ''        | Additional CSS classes to apply to the avatar.      |

## Sizes

The Avatar component supports three sizes:

- `small`: Displays a small avatar (width and height of 32px).
- `medium` (default): Displays a medium-sized avatar (width and height of 48px).
- `large`: Displays a large avatar (width and height of 64px).

## Additional Props

- `className`: Allows passing additional CSS classes to the avatar for custom styling.
